# 이미지 콜라주 프로젝트

40장의 사진을 예쁘게 정리해서 한 장의 콜라주로 만들고,  
마지막에 “공주님 최고” 같은 멋진 한글 워터마크까지 얹어주는 프로젝트예요

## 폴더 구조
이미지_콜라주/
├── img/                # 여기에 원본 사진 40장 (이름 제각각 .jpeg 등)
├── input/              # (필요하면 쓰는 중간 폴더)
├── output/             # 최종 결과물들이 여기로 쏙쏙 저장됨
├── step_1_1.py         # IMG_DIR, IN_DIR, OUT_DIR 경로 설정 + 폴더 자동 생성
├── step_1_2.py         # (테스트용) 이미지 열어보고 글자 넣어보기
├── step_xxx.py         # 중간 과정들
└── README.md           # 이 파일!
text## 지금까지 한 일 (순서대로)

1. **img 폴더 자동 생성**  
   `step_1_1.py` 실행 → img, input, output 폴더 자동으로 만들어짐

2. **40장 사진 이름 정리**  
   img 폴더 안의 모든 .jpeg/.JPEG 파일을  
   `img_001.jpg` ~ `img_040.jpg` 로 깔끔하게 일괄 변경 (Jupyter에서 실행)

3. **한 장씩 열어보고 테스트**  
   Pillow로 이미지 열기, `img.show()`, 글자 넣어보기 등 무수히 시도  
   (폰트 깨짐, 글자 안 보임, 한글 네모 등 전쟁 치름…)

4. **8장 가로 콜라주 테스트**  
   500×500으로 잘라서 1×8 가로로 붙여봄 → 성공!

5. **40장 전체 콜라주 완성**  
   5행 × 8열 = 40장 완벽 콜라주 제작 (아마 step_3_2.py 근처)

6. **최종 마무리 – 한글 워터마크 얹기**  
   왼쪽 위에 반투명 검정 박스 + 크고 선명한 한글 글자 (“공주님 최고” 등)  
   → macOS 시스템 폰트(AppleGothic)로 한글 깨짐 완벽 해결  
   → 폰트 없어도 자동으로 나눔고딕 다운로드까지 되는 안전장치 완료
